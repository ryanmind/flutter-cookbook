2. **Dart OOP** → class, inheritance, mixin
3. **Dart Async** → Future, Stream, async/await
4. **State Management** → setState → Provider → Riverpod
5. **Navigation** → Navigator → GoRouter
6. **Networking** → http/dio + JSON parsing
7. **Storage** → SharedPreferences/Hive/SQLite
